GeoDB Cities API

This page uses JavaScript fetch() to send a GET request to an external public API and display city data.

JavaScript status will appear here.

External API

City search

API endpoint: GET /v1/geo/cities. Enter your RapidAPI key, choose filters and load cities.

How it works

What JavaScript does

  1. Reads form values: country, prefix, population and limit.
  2. Creates URL parameters with URLSearchParams.
  3. Sends fetch() request to GeoDB Cities API.
  4. Converts response with response.json().
  5. Renders the received cities on the page.

For real API loading you need an active RapidAPI key. The demo button is added only to show the same rendering format without a key.

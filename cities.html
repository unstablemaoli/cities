How it works

What JavaScript does

  1. Reads values from the form.
  2. Creates URL parameters with URLSearchParams.
  3. Sends a fetch() request to GeoDB Cities API.
  4. Converts the response using response.json().
  5. Builds city cards and inserts them into the DOM.

The demo button shows the same interface without an API key. The real request needs an active RapidAPI key.
